## 6 아키텍처 설계와 상세 설계
### 6.1 아키텍처 설계 : 필요성
> 아키텍처 : 특성을 결정짓는 기본 구조를 일컫는 말

- SWE
    + 동적 아키텍처 : 실행 구조  
    &rarr; C/S, Web, Mobile, IoT, 계층, MVC 등
    + 정적 아키텍처 : SW 소스코드 구조  
    &rarr; 시스템, 부시스템, 컴포넌트, 모듈, 패키지, 클래스 등

**설계 단계에서 필요한 작업**
1. 동적 구조 설계 [ 실행 ]
1. 정적 구조 설계 [ 소스코드 구조 ]
1. DB 구조 설계 [ 데이터 ]
1. UI 설계 [ 외관 ]
